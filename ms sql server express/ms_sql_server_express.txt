Установка.

    sql server

    sql server manager studio

Соединение с другого компьютера.

    Открыть Диспетчер конфигурации SQL Server.

        SQL Server 2016 - C:\Windows\SysWOW64\SQLServerManager13.msc

        SQL Server 2014 (12.x) - C:\Windows\SysWOW64\SQLServerManager12.msc

        SQL Server 2012 (11.x) - C:\Windows\SysWOW64\SQLServerManager11.msc

        SQL Server 2008 - C:\Windows\SysWOW64\SQLServerManager10.msc

    В разделе Сетевая конфигурация SQL Server щелкнуть элемент "Протоколы для".

    В списке протоколов щелкните правой кнопкой мыши протокол, который необходимо включить (TCP/IP), и выберите команду Включить.

    При подключении к этому экземпляру из другого компьютера необходимо открыть порт соединения в брандмауэре.

    Прежде чем появится возможность открыть порт в брандмауэре, необходимо предварительно настроить компонент Компонент Database Engine для прослушивания конкретного порта.

    На правой панели дважды щелкните TCP/IP.

    В диалоговом окне Свойства TCP/IP перейдите на вкладку IP-адреса.

    Введите доступный номер порта (на моём компьютере) в поле TCP-порт раздела IPAll (фиксированного порта). Для того чтобы посмотреть доступные порты вводим в консоль netstat -a.

    На левой панели щелкните Службы SQL Server.

    На правой панели щелкните экземпляр SQL Server правой кнопкой мыши и выберите пункт Перезапустить. После перезапуска компонент Компонент Database Engine будет принимать соединения через заданый порт.

Открытие портов в брандмауэре.

    На левой панели Брандмауэр Windows в режиме повышенной безопасности щелкните правой кнопкой мыши раздел Правила для входящих подключений и выберите на панели действий пункт Создать правило.

    В диалоговом окне Тип правила выберите Порт и нажмите кнопку Далее.
    
    В диалоговом окне Протокол и порты выберите протокол TCP. Выберите Определенные локальные порты и введите номер порта. В диалоговом окне Действие выберите Разрешить соединениеи нажмите кнопку Далее.

    В диалоговом окне Действие выберите Разрешить соединениеи нажмите кнопку Далее.

    В диалоговом окне Профиль выберите профили, описывающие среду соединения компьютеров, который нужно подключить к компоненту Компонент Database Engine, и нажмите кнопку Далее.

    В диалоговом окне Имя введите имя и описание для этого правила, а затем нажмите кнопку Готово. (test_automarshall)

Соединение с сервером с другого компьютера.

    Открыть среду Management Studio.

    В диалоговом окне Соединение с сервером выбираем в поле тип сервера "Ядро СУБД" или Database Engine.

    В поле Имя сервера введите tcp:<ip_компьютера>,порт
