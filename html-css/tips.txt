- если нужно задать размер, который не должен изменяться, то задаём например максимальную ширину (max-width: 200px)

- для выравнивания элементов нужно использовать display: flex

- ширину и высоту лучше задавать в px, когда нужно 100% соответствие с макетом, в остальных случаях лучше использовать %

- не использовать рядом px и %

- лучше задавать только высоту для элемента

- чтобы посчитать высоту элемента нужно взять высоту контейнера и сложить с суммой отступов элемента от данного контейнера

- margin лучше чем padding, потому что если например понадобится покрасить

- именование

    - блок максимум с 3-мя словами: slider-menu-row

    - если вложенность блока в блок больше 3-х слов, то разбиваем slider-menu-row-link на class="slider-menu-row" и class="row-link", то есть берём последнее слово первого блока и начинаем название второго с него
